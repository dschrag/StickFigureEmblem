Credits

Stick Figure Emblem is the final project of Derek Schrag and Daniel Miller for CS 252, Fall 2015

Stick Figure Emblem, as the name implies, is based off of the popular Nintendo series, Fire Emblem.

Special thanks to Dana Muenter for our stick figure models.

Music written by Eric Matyas, from Sound Image

Stick Figure Emblem is hosted on IBM Bluemix. The server is written in Node.js. The webpage uses Bootstrap. The game engine was written in Three.js. The scores database is hosted on the NoSQL-based CloudantDB.
